RxJS (Reactive extensions for JavaScript) is a library for composing asynvhronous and event-based programs by using observable sequences.
It provides one core type: Observable, satellite types: Observer/Schedulers/Subjects, Operators: [map, filter, reduce, every, etc..] (inspired by Array methods)
RxJS library enables developers to work with asynchronous data streams easily. The core concept of RxJS revolves around observables, which are representations of any set of values over any amount of time. These values can be events, user inputs, HTTP responses, or any other asynchronous data source. Observable then emit this data, which is consumed by subscribe() method in required components, and then we can RxJS operators for manipulation.
